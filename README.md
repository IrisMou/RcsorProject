# 尔莱斯电竞-专业电竞配件平台

  应用介绍: "尔莱斯电竞-专业电竞配件平台"app是一款购买电竞配件(电竞杯子、电竞耳机、电竞饰品和电竞鼠标垫)的应用。

  应用功能: 
  
    商品分类：电竞杯子、电竞耳机、电竞饰品、电竞鼠标垫等类型商品。
    下单方式：直接下单结算、假如购物车批量结算。
    购买流程：线上下单，商家免费配送，货到付款。
    福利中心：下单完成获得对应数量积分，积分可直接在福利中心兑换商品。
    我的订单：商品下单的订单和积分兑换的商品，均在我的订单列表。
